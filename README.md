# C-Learning-Journey 🚀

Welcome to my personal C programming journey repository! This project is where I practice, document, and improve my C programming skills through notes, exercises, and small applications.

## 🎯 Purpose
This repository helps me:
- learn the basics of C programming,
- practice problem-solving and coding logic,
- build simple projects step by step, and
- keep track of my progress while studying.

## 📁 Repository Structure
```text
C-Learning-Journey/
├── Documentation/         # Learning notes, slides, and theory material
├── Exercises/             # Practice problems and coding exercises
├── Projects/              # Small complete C programs/projects
│   ├── Average Tracker/    # Calculates average scores of students
│   └── Calculator/         # Basic arithmetic calculator
└── README.md              # Main repository overview
```

## 📚 What’s Included

### Documentation
This folder contains study material such as notes and presentations related to C concepts.

### Exercises
This folder is for hands-on practice and small coding tasks.

### Projects
This folder contains complete beginner-friendly C programs and mini applications.

## 💡 Learning Goals
- understand C syntax and program structure
- practice variables, loops, and conditions
- work with input/output operations
- build small real-world programs

## ▶️ How to Use
1. Open the folder you want to explore.
2. Read the related notes or project documentation.
3. Compile and run the C files using a compiler such as `gcc`.

Example:
```bash
gcc program.c -o program
./program
```
